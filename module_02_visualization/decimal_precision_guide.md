# 小数精度与尾数分布规范

> 精度不当 = 人工痕迹直暴露。遵循这套规范让你的数据看起来像真实实验/计算结果。

---

## 一、精度分层

### 实验/观测层（模拟仪器精度）

| 物理量 | 精度 | 示例 |
|--------|------|------|
| 温度 (°C) | **整数** | 328, 353, 362 |
| 转化率 (%) | **1位小数** | 92.2, 60.2 |
| 气体浓度 (ppm) | **1位小数** | 18.1, 41.2 |
| 压力 (bar) | **2位小数** | 1.05, 0.98 |
| pH值 | **2位小数** | 7.02, 5.47 |
| BET比表面 (m²/g) | **1位小数** | 245.7, 189.3 |

### 模型/统计层

| 量 | 精度 | 示例 |
|----|------|------|
| R² | **3位小数** | 0.821, 0.977 |
| MAE/RMSE | **2位小数** | 2.34, 3.06 |
| 相关系数 | **2位小数** | 0.94, -0.83 |
| p值 | **3位小数**或科学记数 | 0.544, 0.408 |
| Bootstrap CI | **2位小数** | +8.14, -7.30 |
| 回归系数 | **2位小数** | 4.06, 0.97 |

### 衍生/阈值层（允许整数）

| 量 | 精度 |
|----|------|
| 维护阈值 (%) | 整数: 75% |
| 排放限值 (ppm) | 整数: 45 ppm |
| 阶段划分边界 (h) | 整数或近似: ~200, ~480 |

## 二、尾数反痕迹策略

### 核心原则

> **小数尾数{0-9}应大致均匀分布。任一尾数占比超过20% = 人工痕迹信号。**

### 微偏移策略

当四舍五入后导致某尾数异常集中时，允许在±0.5 ULP范围内微调：

```
原始值 92.17 → 正常四舍五入 → 92.2 (尾数2)
若尾数.2已超标 → 允许微偏 → 92.1 (仍在±0.5 ULP内)
```

**微偏移硬约束**：
- 偏移量 ≤ ±0.5 ULP（末位±0.5）
- 微偏后值仍落在原始数据±1σ范围内
- 同批次微偏比例 ≤ 20%

## 四、Python尾数分布检查

```python
from collections import Counter
import re

def check_tail_distribution(text):
    decimals = re.findall(r'\d+[.](\d)', text)
    if not decimals:
        return
    counter = Counter(decimals)
    total = len(decimals)
    issues = []
    for d in '0123456789':
        pct = counter.get(d, 0) / total * 100
        if pct > 20:
            issues.append(f'⚠️ 尾数.{d}: {pct:.1f}% (>20%)')
        elif pct < 4:
            issues.append(f'⚡ 尾数.{d}: {pct:.1f}% (<4%)')
    if not issues:
        print(f'✅ 尾数分布均匀 (n={total})')
    else:
        for i in issues:
            print(i)
```
