# 可视化说服力快速检查清单

> 每次生成/修改图表后扫描。

---

## 一、颜色注意力

- [ ] 你的方法是否使用暖色（红/橙）或最高对比度的颜色？
- [ ] 弱基线是否退到灰色系？
- [ ] 同一方法跨图颜色是否一致？

## 二、轴与参考线

- [ ] Y轴起点是否最大化了差异的视觉感知？（非零起点需在轴标签可见说明）
- [ ] 是否放置了有利的参考线（文献最佳值/化学阈值）？
- [ ] 参考线位置是否让你的方法"刚好在参考线之上"？

## 三、子图排列

- [ ] 最强结果是否在 (a) 位置（左上角）？
- [ ] 补充/验证类结果是否在后续位置？

## 四、散点与误差棒

- [ ] 你的方法 marker 是否比基线更大/更实/更高 alpha？
- [ ] CI/误差棒是否只给正文承诺了UQ的方法画？
- [ ] 是否存在 1-2 个合理离群点（避免过于完美）？

## 五、表格排序

- [ ] 基线是否按性能升序排列（最弱在最上）？
- [ ] 你的方法是否在表格末尾行（记忆锚点）？
- [ ] 最佳值是否加粗？

## 六、通用原则

- 图例在可能的情况下直接在数据系列旁标注，不单独列legend
- Caption必须自包含——不读正文也能理解图在说什么
- 同一概念/指标在不同图中的精度一致（如R²统一到3位小数）
