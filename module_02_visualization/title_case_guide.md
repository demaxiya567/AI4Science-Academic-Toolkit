# 论文标题大小写规范

> 按目标期刊类型选择 Sentence case 或 Title case。

---

## 按期刊类型判定

| 期刊家族 | 大小写规则 | 示例 |
|---------|-----------|------|
| **MDPI** (Catalysts等) | **Sentence case** | `Asynchronous activity window evolution in VWT SCR catalysts` |
| **ACS** (JACS, ACS Catalysis) | **Sentence case** | 同上 |
| **RSC** (Catal. Sci. Technol.) | **Sentence case** | 同上 |
| **Elsevier** (J Catal, Appl Catal B) | **Sentence case** | 同上 |
| **Nature** 系列 | **Title case** | `Asynchronous Activity Window Evolution in VWT SCR Catalysts` |
| **Science** | **Title case** | 同上 |

## Sentence case 细则

- **首字母大写**
- **专有名词保留大写**：SCR, DFT, VWT, Bootstrap, DOE
- **化学式/缩写保留**：V₂O₅, WO₃, TiO₂, NH₃, XPS, XRD
- **其余全部小写**：activity, window, evolution, under, revealed by

## Title case 细则

- 名词/动词/形容词/副词首字母大写
- 冠词(a/an/the)、介词(in/of/under/by)、连词(and/or/but)小写
- 专有名词/缩写始终大写

## 反例

| ❌ 错误 | ✅ 正确 |
|---------|--------|
| `Asynchronous Activity Window Evolution` (MDPI) | `Asynchronous activity window evolution` |
| `A Novel Framework for Catalyst Screening` (ACS) | `A novel framework for catalyst screening` |
