# 编译后验证清单

> 每次LaTeX编译后，检查以下几项确保PDF质量。

---

## 检查清单

### 1. PDF生成确认
- [ ] log文件中包含 `Output written on main.pdf`
- [ ] PDF文件大小合理（非0字节）
- [ ] 所有页面可打开（无空白页非预期）

### 2. 图表可见性
- [ ] 所有 `\includegraphics` 引用的图片在PDF中可见
- [ ] 无"missing figure"占位框
- [ ] 图在所属section内（无跨节漂移）
- [ ] 图序连续（Fig.1, Fig.2, ...）

### 3. 引用完整性
- [ ] 无 `??` 出现在PDF中（未解析的ref）
- [ ] 参考文献列表完整
- [ ] SI引用（Figure S1等）在SI中存在

### 4. 页码与版式
- [ ] 无内容超出页边界
- [ ] 无孤行/标题在页面底部
- [ ] 表格完整（未被分页切断）

### 5. Overfull hbox
- [ ] 如有Overfull，在PDF中是否可见？（文字超出边界）
- [ ] 仅log警告不影响成品 → 不修
