# 投稿成熟度快速评估

> 回答"够了没"和"下一步做什么"，不回答"做什么"。

---

## 核心问题

1. **最低底线**：这篇论文有没有任何会让编辑直接desk reject的问题？
   - [ ] 虚假引用
   - [ ] 正文-SI数值矛盾
   - [ ] 严重逻辑矛盾
   - [ ] 图序/图位置错乱

2. **核心价值**：这篇论文的核心化学洞见是否能让审稿人觉得"有意思"？
   - [ ] 有明确的新发现/新维度/新机制
   - [ ] 不是重复已知规律
   - [ ] 有可验证的预测或实验指导

3. **写作质量**：是否达到了目标期刊的写作标准？
   - [ ] AI痕迹不显著
   - [ ] 逻辑清晰
   - [ ] 叙事流畅

4. **边际收益**：继续修改还能提升多少accept概率？
   - [ ] 还有明显短板 → 先补短板
   - [ ] 短板已补，只剩润色 → 可以投了
   - [ ] 已经改过3轮+ → 不要无限优化

---

## 投稿建议

| 状态 | 建议 |
|------|------|
| 全部通过 | ✅ 可以投稿 |
| 只有写作问题 | ⚠️ 花1天润色后投 |
| 有1个数值矛盾 | 🔴 必须修后再投 |
| 有虚假引用 | 🔴 禁止投稿 |
| 已改过3轮 | ✅ 停止修改，投 |

---

## 一句话执行口令

> **不存在致命错误 → 核心洞见清晰 → 写作过得去 → 投。不要等完美。**
