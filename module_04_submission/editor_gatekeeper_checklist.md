# 编辑速览模拟器（Editor Desk Review）

> 模拟期刊编辑5分钟desk review决策。

---

## 编辑的阅读顺序

```
1. Title（3秒）→ 有没有化学发现的味道？
2. Abstract第1句（5秒）→ 问题是什么？
3. Abstract最后1句（5秒）→ 发现了什么？
4. Fig.1（10秒）→ 独立传达核心发现？
5. Results标题扫描（10秒）→ 含化学信息？
6. 扫一眼图表质量（10秒）→ 专业度？
7. 参考文献（5秒）→ 引用近期文献？
```

---

## 检查清单

### 1. Title（3秒判定）
- [ ] 主语是化学现象/发现，不是算法/框架名？
- [ ] 读者能在1秒内判断"这是什么类型的研究"？
- [ ] 避免 "A novel framework..." / "A new approach..."

### 2. Abstract（10秒判定）
- [ ] 第一句是化学问题urgency，不是"we developed"？
- [ ] 最核心的发现是在中间位置展开，不是一开始就扔出最强结果？
- [ ] 最后一句是化学预测/意义，不是R²数值？
- [ ] 没有 "outperforms" / "novel" 等自夸词？

### 3. Fig.1（10秒判定）
- [ ] 不看caption能大概知道在说什么？
- [ ] 核心发现是图上的主角（不是被挤到角落）？
- [ ] 配色专业？非默认jet/rainbow？

### 4. 图表总体印象（10秒）
- [ ] 所有图风格统一（字体/配色/线宽）？
- [ ] 无低分辨率/模糊的图？
- [ ] 无过于复杂的图（太多子图挤一页）？

### 5. 参考文献扫描（5秒）
- [ ] 引用了目标期刊近期文章？
- [ ] 引用密度合理（Intro每段≥3个）？

---

## 判定

| 结论 | 含义 | 概率 |
|------|------|------|
| ✅ Send for review | 送审 | 编辑感兴趣 |
| ⚠️ Maybe | 边界案例 | 可能再读一段或送副编辑 |
| ❌ Desk reject | 直接拒 | 以上5项有2+项不达标 |
