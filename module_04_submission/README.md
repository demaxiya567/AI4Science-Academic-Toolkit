# Module 4: 投稿前全流程Pipeline

> **5分钟致命错误检查 + 全流程11步审查 + 编辑速览模拟**

## 核心理念

投稿前最怕的不是"某处写得不够好"，而是"存在致命错误没发现"——虚假引用、正文SI数值矛盾、图漂移到错误章节、作者信息缺失。这些错误一旦被编辑发现，直接desk reject。本模块提供从快速扫描到全流程pipeline的投稿前保障。

## 包含内容

| 文件 | 用途 |
|------|------|
| `submission_pipeline.md` | 完整11步投稿前流程（含对抗循环） |
| `pre_submit_5min_checklist.md` | 5分钟快速致命错误扫描 |
| `editor_gatekeeper_checklist.md` | 编辑desk review决策模拟 |
| `submission_readiness_checklist.md` | 投稿成熟度快速评估 |

## 投稿前流程概览

```
Step 0: 文件确认 — 确认投稿模式和文件清单
Step 1: 编辑速览模拟 — 编辑5分钟决策
Step 2: 审稿人模拟审查 — 同行评审视角
Step 3: AI写作痕迹检测
Step 4: 数值一致性检查
Step 5: 参考文献审计
Step 6: 图表技术验证
Step 7: 逻辑连贯性审校
Step 8: Go/No-Go决策
```

## 5分钟紧急扫描

时间紧迫时，至少执行 `pre_submit_5min_checklist.md` 中的5类致命错误检查。
