# 投稿前5分钟致命错误检查

> 最高频的 desk reject / post-acceptance 撤稿原因猎杀。
> 目标：5分钟内完成，只报告会直接导致拒稿的问题。

---

## 🔴 类型1：虚假引用（最高风险）

**检查范围**（不做全面检查，只查高风险引用）：
- 小众期刊 + 不常见作者组合
- 缺少DOI的引用
- 2023-2026年的引用（LLM幻觉高发区）

**执行**：
1. 从bib中提取所有引用
2. 对上述特征引用用搜索引擎验证：`"作者姓" "标题关键词" 年份`
3. 找不到 → 标记为 🔴 手动核查
4. 找到但信息不匹配 → 标记为 🔴 修正

---

## 🔴 类型2：正文-SI数值矛盾

**快速检查**：
```bash
# R²/T50/MAE在正文和SI中的值是否一致
grep -n "R\\^2" main.tex SI.tex | sort
grep -n "T_{50}" main.tex SI.tex | sort
```

**常见矛盾**：
- 正文说 R²=0.92，SI中同数据算出来0.89
- 正文说 T50=180°C，SI图中读出来175°C

---

## 🔴 类型3：图表问题

- [ ] 图序是否连续？（不跳号，无重复编号）
- [ ] 图是否在正确的section中？（编译PDF逐页检查）
- [ ] 所有图在PDF中可渲染？（无缺失、无乱码）
- [ ] Caption是否自包含？（不依赖正文）

---

## 🔴 类型4：作者/单位/基金信息

- [ ] 所有作者姓名拼写正确
- [ ] 单位名称与标准名称一致
- [ ] 通讯作者邮箱正确
- [ ] 基金项目编号准确
- [ ] 利益冲突声明已包含

---

## 🔴 类型5：格式致命问题

- [ ] 参考文献格式与目标期刊一致
- [ ] 正文字体/行距/页边距符合模板
- [ ] SI有独立标题页（如期刊要求）
- [ ] 所有文件命名规范（无空格/中文）

---

## 执行顺序

```
1. 类型1（虚假引用） → 最致命，优先
2. 类型2（数值矛盾） → 审稿人必查
3. 类型3（图表问题） → 编辑一眼可见
4. 类型4（作者信息） → 简单但致命
5. 类型5（格式问题） → 快速过一遍
```
