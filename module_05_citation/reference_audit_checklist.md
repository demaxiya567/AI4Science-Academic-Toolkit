# 投稿前参考文献审计清单

> 重点检查LLM幻觉高发区，不做100%全面检查（可能反而徒增不必要风险）。

---

## 高风险引用类型

| 类型 | 风险 | 检查方法 |
|------|------|---------|
| 2023-2026年发表 | 🔴 高 | LLM幻觉高发区，逐条核实 |
| 小众期刊+不常见作者 | 🔴 高 | 搜索引擎验证 |
| 缺少DOI | 🟠 中 | 尝试在Crossref查找 |
| 作者名拼写异常（拼音变体） | 🟠 中 | 查作者Google Scholar页面 |
| 多篇论文元素混搭引用 | 🟠 中 | 检查标题是否匹配作者 |

---

## 检查清单

### 必需检查
- [ ] 所有引用在真实文献库中存在（DOI可解析或Google Scholar可找到）
- [ ] 无重复引用（同一文献不同key）
- [ ] BibTeX key命名一致（建议 `authorYYYY` 格式）
- [ ] 目标期刊要求的引用格式

### 建议检查
- [ ] 自引确认准确
- [ ] 期刊官方缩写与标准一致
- [ ] 链接型引用（URL/Dataset DOI）可访问
