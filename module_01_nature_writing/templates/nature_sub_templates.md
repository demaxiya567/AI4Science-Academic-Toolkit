# Abstract写作模板

> 记住：编辑先看Abstract结论句。那是你最重要的25个词。

```
[化学问题的urgency] __________________________________________
[现有方法局限] ________________________________________________
[方法一句话] Here we use [方法简称] to [化学发现] _____________
[核心发现1] __________________________________________________
[核心发现2] __________________________________________________
[可验证预测] This suggests that ________________________________
[意义升华] ____________________________________________________
```

# Introduction写作模板

> 不罗列文献，引导读者确认"这里有一个真缺口"。

### P1：化学问题重要性
```
[化学/环境/能源问题] is critical because [具体场景/数字].
Despite decades of research, [具体瓶颈] remains unresolved.
```

### P2：实验方法局限
```
Conventional experimental approaches face fundamental limitations:
[清单，不超过3个]. In particular, [最关键的1个局限].
```

### P3：已有AI/计算方法不足
```
Recent computational efforts have aimed to address this gap through
[方法类型]. However, these approaches [没揭示什么化学信息].
A key limitation is that [方法学局限].
```

### P4：我们的方法
```
Here, we introduce [方法简称], which [为什么能突破].
Unlike prior methods, it [独特优势].
```

### P5：发现预告
```
Here we show that [核心结构/分区/排序].
Specifically, [具体证据1] reveals [发现1],
while [具体证据2] identifies [发现2].
These findings establish [化学意义].
```

# Results段落内部结构模板

```
Opening: To determine [化学问题]...

Body: [方法/Analysis] reveals [核心发现] (Fig. X).
[数据/数值嵌入] — [具体数字] compared to [基准].

Interpretation: This indicates that [化学含义].
The [方向/趋势] suggests [机制/规律].

Bridge: This raises the question of [下一问题],
which we examine through [下个分析].
```

# Discussion结构模板

```
D1: [核心发现的深层含义]
    "This observation is consistent with [文献现象],
     and further suggests that [新的化学假设]."

D2: [与已知知识的关系]
    "Extending previous observations, our results show that [新维度]."

D3: [局限+边界]
    "Within the [体系/条件范围] tested, [核心发现] holds.
     Extension to [更广范围] warrants further investigation."

D4: [展望+可验证预测]
    "This predicts that [具体实验] would yield [具体结果],
     which could be validated by [具体表征方法]."
```
